Timesplitters 2 HD Fonts

Overview

"TS2 HD Fonts" is a resource pack for Timesplitters 2, intended to be used in the Dolphin Emulator.

This mod replaces textures for fonts and other selected elements of game's user interface with higher resolution recreations for a more pleasant experience on modern displays.

Features

  • Font textures remade using original fonts
  • Aspect ratio corrected loading screens
  • Higher quality developer logos and title
  • Redrawn health bar, radar and other miscellaneous UI elements
  • Optionally downloadable button icons for Dualshock prompts
