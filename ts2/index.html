Timesplitters 2 HD Fonts

Overview

"TS2 HD Fonts" is a resource pack for Timesplitters 2, intended to be used in the Dolphin Emulator.

This mod replaces textures for fonts and other selected elements of the game's user interface with higher resolution recreations for a more pleasant experience on modern displays.

Features

  • Font textures remade using original fonts
  • Aspect ratio corrected loading screens
  • Higher quality developer logos and title
  • Redrawn health bar, radar and other miscellaneous UI elements
  • Optionally downloadable button icons for Dualshock prompts

Installation

  1. Download the resource pack
  2. Open Dolphin's user folder (located in "Users\%username%\Documents\Dolphin Emulator\" on Windows by default) and open the "ResourcePacks" folder in it
  3. Put the .zip file inside it
  4. Open Dolphin Emulator and locate the "Resource Pack Manager" in the "Tools" tab
  5. In "Resource Pack Manager" select the mod and press "Install"
