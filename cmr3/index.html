colin mcrae rally 3 HD UI

Overview

"Colin McRae Rally 3 HD UI" is a mod for the PC version of Colin McRae Rally 3.

This mod replaces textures for fonts and most other elements of the game's user interface with higher resolution recreations for a more pleasant experience on modern displays.

Features

  • Made with 4K resolution in mind
  • Font atlases remade using original fonts
  • Support for EFIGS, Polish and Czech languages
  • Banners redrawn for every stage
  • Majority of in-game UI elements replaced
